# URL Status Checker

A Python script that checks whether a website is reachable.

## Description
This project sends a request to a given URL and returns the HTTP status code if the website is reachable.

## Features
- URL checking
- Displays HTTP status code
- Simple network request handling

## Technologies Used
- Python
- urllib module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python url_checker.py

4. Enter a URL when prompted

## Example

Input:
https://google.com

Output:
Status: 200 (Website is reachable)
